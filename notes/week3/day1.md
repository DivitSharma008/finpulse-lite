# Day 1 — Risk Metrics Theory

- Studied key risk and performance metrics:
  - **Sharpe Ratio** — return per unit of risk taken
  - **Max Drawdown** — largest peak-to-trough decline in portfolio value
  - **Volatility** — standard deviation of daily returns
  - **Risk-Adjusted Return** — return relative to the risk incurred
- Started `metrics.py` and implemented the total return calculation.
